# 路由

### 管理后台

访问后台路由将会自动在所有链接上默认增加 admin_path 路径前缀

例如 `http://your-domain.com/{admin_path}/manage-something`


### 应用后台

访问应用中心将会自动在所有链接上默认增加 extension_admin_path 路径前缀

例如 `http://your-domain.com/{extension_admin_path}/something`

### 前台

默认路由配置，无前缀
